# Границы MVP

## Пользователь получает

- одну веб-ссылку, открывающуюся на телефоне и ПК;
- естественный перевод на американский английский;
- сохранённые профили взрослых персонажей;
- настройку контекста, тона, сленга, флирта, вульгарности, сексуального напряжения и прямоты;
- выбор доступной Qwen-модели;
- быстрое копирование и повторную генерацию;
- перенос персонажей через JSON;
- подключение телефона через QR-код;
- запуск локальной части одним файлом.

## В MVP не входит

- постоянный адрес API;
- работа при выключенном компьютере;
- синхронизация профилей между устройствами;
- аккаунты пользователей и облачная база;
- хранение истории переводов;
- локальная модель через llama.cpp;
- DeepSeek, Kimi или GLM;
- потоковая генерация;
- автоматический обход CAPTCHA;
- автоматическое снятие ограничений Qwen Chat.

## Критерии готовности

1. GitHub Pages открывается с телефона.
2. `start_translator.bat` запускает три локальных компонента.
3. Quick Tunnel создаётся без открытия портов роутера.
4. QR-код подключает сайт к текущему туннелю.
5. Gateway не принимает запрос без правильного токена.
6. Gateway не принимает запрос с чужого origin.
7. Профиль младше 18 лет не сохраняется и не принимается сервером.
8. Результат возвращается одной готовой английской фразой.
9. Секреты отсутствуют в репозитории и коде сайта.
10. При сбое пользователь получает понятную ошибку и лог.
